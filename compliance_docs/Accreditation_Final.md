# 1. Intended Use of Services & Crypto Integration

Paypee is a next-generation financial operating system designed to bridge the gap between traditional fiat currencies and digital assets for African businesses and individuals.

## Intended Use
Paypee intends to use the services provided by **Bitnob** and **Quidax** to enhance institutional liquidity and settle cross-border transactions. Our use cases include:
*   **Cross-Border Remittance:** Utilizing stablecoin liquidity to settle transactions instantly.
*   **Treasury Management:** Holding and managing corporate assets in USD-pegged stablecoins.
*   **Virtual Card Funding:** Powering global spending via crypto-to-fiat conversion layers.

## Crypto Functionalities
*   **Stablecoin Settlement:** Conversion between NGN and USDT/USDC for liquidity.
*   **Lightning Network:** Leveraging Bitnob for micro-payments.
*   **MPC Custody:** Institutional-grade security via partners.

# 2. User Journey Overview

## Onboarding
1.  **Level 1:** Email/Phone verification.
2.  **Level 2:** ID Verification (NIN/Passport/CAC) via **Prembly**.
3.  **Level 3:** Biometric Liveness match via real-time selfie.

## Funding & Transactions
*   **Funding:** Automated NGN virtual accounts via **Fincra**.
*   **Transactions:** Instant FX swaps, Global Payouts, and Virtual Card issuance.
*   **Withdrawals:** Automated settlement to local Nigerian banks or corporate stablecoin wallets.
